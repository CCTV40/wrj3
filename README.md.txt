# 无人机通信心跳监测可视化

## 项目简介
本项目基于 Python + Streamlit 实现无人机通信“心跳”监测系统，模拟无人机每秒发送一次心跳包，实现掉线检测（3秒未收到心跳则报警），并通过可视化界面展示心跳时序变化。

## 核心功能
- ✅ 模拟无人机心跳包（包含序号、时间戳）
- ✅ 掉线检测：3秒未收到心跳自动报警
- ✅ 实时折线图展示心跳序号随时间变化
- ✅ 支持本地运行与 Streamlit Cloud 云端部署
